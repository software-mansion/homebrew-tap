# Software Mansion Homebrew Tap

Homebrew tap for Software Mansion tools.

## Add This Tap

```bash
brew tap software-mansion/tap
```

## Available Apps

- SimCam

## Install SimCam

After adding the tap:

```bash
brew install --cask simcam
```

Or install it in one command:

```bash
brew install --cask software-mansion/tap/simcam
```
